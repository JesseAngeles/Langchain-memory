# LangChain Memory

Conexión a GPT utilizando LangChain con memoria (historial de conversación) como contexto.

## Tecnologías

- TypeScript
- LangChain
- OpenAI API

## Instalación

```bash
npm install
```

Crear un archivo `.env` con tu API key:

```
OPENAI_API_KEY=tu_api_key
```

## Ejecución

```bash
npm start
```
